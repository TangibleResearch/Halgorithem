How it works

A simple source-to-claim pipeline.

Halgorithem cleans source text, splits it into sentences, builds chunks, extracts tokens and numbers, then compares every AI claim to the truth data.

1

Clean

Normalize text, whitespace, quotes, and punctuation.

2

Chunk

Group nearby sentences with overlap for better context.

3

Compare

Score token overlap, important terms, and string similarity.

4

Flag

Return supported, hallucination, or contradiction results.

Install

Run it locally.

These are the same install steps from the project README.

1. Create a virtual environment
python -m venv venv
2. Activate the virtual environment
source venv/bin/activate
3. Install requirements
pip install -r requirements.txt
4. Run Halgorithem
python main.py
